## About 

This project is a modern web application built using Laravel for the backend, Vite for fast bundling, React for a dynamic frontend, Tailwind CSS for responsive and customizable styling, and Inertia.js to seamlessly connect the frontend and backend.

- [Laravel](https://laravel.com/docs/12.x/installation).
- [Vite React Installation](https://laravel.com/docs/12.x/vite#react).
- Server side [Intertia](https://inertiajs.com/docs/v2/installation/)
- Client side [Intertia](https://inertiajs.com/docs/v2/installation/client-side-setup).
- [Tailwind CSS](https://tailwindcss.com/docs/installation/using-vite).
